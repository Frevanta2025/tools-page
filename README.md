# Mireflecta Tools

Simple local-first tools to structure your files, notes, and workflows.

## Overview

Mireflecta Tools is a collection of lightweight, privacy-first utilities designed to help you organize digital information quickly and reliably.

All tools run entirely in your browser. No files are uploaded, no accounts are required, and no data is stored externally.

## Available Tools

### Markdown Merger
Combine multiple Markdown files into a single structured document.

→ https://md.mireflecta.com

### File Structurer
Turn messy files into clean, consistently named and structured batches.

→ https://fs.mireflecta.com

### Folder Maker
Generate ready-to-use folder structures with localized naming and index files.

→ https://folders.mireflecta.com

## Principles

- **Local-first** — all processing happens in your browser  
- **No uploads** — your files never leave your device  
- **No accounts** — no sign-up, no tracking  
- **Transparent** — source code is available for review  
- **Simple** — focused tools with minimal friction  

## Positioning

These tools are independent utilities that demonstrate a broader approach to structured digital work.

They are part of **Mireflecta**, a system in development for organizing personal and professional information flows.

## Tech Stack

- HTML
- CSS (custom properties for theming)
- Vanilla JavaScript
- No backend
- No external dependencies
